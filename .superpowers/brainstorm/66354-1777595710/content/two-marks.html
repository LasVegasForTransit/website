Two marks · one wordmark · one system

Proposed architecture: a fun mark for small/playful contexts, a bold mark for medium/formal contexts, and the existing wordmark as the type lockup. Each derived from the Welcome sign at different intensities.

The two marks
side by side · same vocabulary, different register
FUN MARK · BULB LETTERS

Each letter in a bulb chip — direct lift from the Welcome sign's "WELCOME" treatment. Reads as Vegas without ever rendering the literal sign. Built for small scales, charm, and personality.
BOLD MARK · DIAMOND + STAR
Diamond + 8-point star + 4 cardinal anchors. Hairline construction, no bulbs, no posts. Reads as civic seal. Built for medium-large scales, authority, and gravitas.

Usage matrix · where each mark belongs
discipline keeps the marks meaningful
Context Use Why
Browser favicon (16/32px) Bold mark Bulb letters become unreadable noise <20px; the diamond+star reads as a single shape
Social avatar (square) Bold mark Square aspect — bulb letters stretch wide; bold mark is naturally square
Email signature, byline Fun mark Inline with text — the chip rhythm pairs well with sans-serif body copy
Site header (left) Wordmark + small fun mark beside it The lockup carries authority; the chips add personality
Footer stamp Fun mark + © line Footers want personality; chips give it without taking space
Stickers, T-shirt chest hit Either (depending on size and shape) Both work; pick by aspect ratio of the surface
Report covers, formal docs Bold mark + wordmark Civic-seal weight pairs naturally with the formal typography
Co-branding (with allied orgs) Bold mark only A symbol-only mark holds its weight beside other logos better than chips
